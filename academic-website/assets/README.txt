ASSETS FOLDER
=============
Put your images and documents here, then reference them from content.yml.

Recommended files:
  profile.jpg           -> your headshot (square, ~600x600px looks best)
  Faust_CV.pdf          -> your CV (linked from the header)
  fieldnote-1.jpg, etc. -> photos for the Field Notes page
  faust-2025-*.pdf      -> publication PDFs you want to host

In content.yml you then write paths like:  assets/profile.jpg

Tips:
- Keep images under ~500 KB so the site stays fast (resize before uploading).
- File names are case-sensitive on GitHub Pages: assets/Photo.JPG != assets/photo.jpg
